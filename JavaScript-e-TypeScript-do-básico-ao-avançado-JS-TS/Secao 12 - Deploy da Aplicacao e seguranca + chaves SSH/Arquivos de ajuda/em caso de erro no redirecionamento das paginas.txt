app.use(helmet({contentSecurityPolicy: false})) pois você configura a falta de ssl do helmet

Passos para enviar alterações ao servidor cloud da sua maquina local:
git add . > para selecionar todos os arquivos
git commit -am 'Mensagem que você quer adicionar'
git push agenda master
------------------------------
entre na pasta que estão os arquivos do projeto no servidor cloud e digite;

Na maquina do servidor;
git pull agenda master > exemplo

cd /etc/nginx/sites-enabled/

sudo systemctl restart nginx > para reiniciar o nginx e aplicar qualquer atualização feita
