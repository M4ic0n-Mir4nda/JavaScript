curl -sL https://deb.nodesource.com/setup_16.x | sudo -E bash -
sudo apt-get install -y nodejs
Obs: Esta versão do nodejs é depreciada, aparentemente nesta versão de ubuntu não é instalada as versões recentes do nodejs. Procurar pesquisar mais para ter certeza disso.
